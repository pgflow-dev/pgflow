## Build AI Agent Workflows in Minutes

Define complex AI workflows with parallel execution and dependencies using a simple, type-safe API:

```typescript
// AI Agent workflow with parallel processing
new Flow<{ userQuery: string }>()
  .step("retrieveContext", async ({ userQuery }) => {
    return await vectorSearch(userQuery);
  })
  .step("generateResponse", ["retrieveContext"], async ({ retrieveContext, run }) => {
    return await callLLM({
      prompt: run.userQuery,
      context: retrieveContext.results
    });
  })
  .step("analyzeSentiment", ["generateResponse"], async ({ generateResponse }) => {
    return await analyzeSentiment(generateResponse.text);
  })
  .step("summarize", ["generateResponse"], async ({ generateResponse }) => {
    return await summarizeWithAI(generateResponse.text);
  })
  .step("saveToDb", ["analyzeSentiment", "summarize", "generateResponse"], async (input) => {
    return await saveResults({
      query: input.run.userQuery,
      response: input.generateResponse.text,
      sentiment: input.analyzeSentiment.score,
      summary: input.summarize
    });
  });
```

## Supercharge Supabase Background Jobs

Create and run workflows with a straightforward process:

1. **Define your workflow** in TypeScript with full type safety and IDE autocompletion
2. **Deploy with one command** - `npx pgflow install` handles migrations and configuration
3. **Run as Supabase Edge Functions** with automatic restarts and zero infrastructure overhead
4. **Start workflows** from your application code, database triggers, or REST APIs
